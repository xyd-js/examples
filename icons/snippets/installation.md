## Installation

::::steps
1. Install the <code>xyd</code> CLI:
:::code-group{title="xyd installation"}
```bash bun
bun add -g xyd-js
```

```bash pnpm
pnpm add -g xyd-js
```

```bash npm
npm i -g xyd-js
```
:::

2. Clone GitHub repo:
```bash [lines]
git clone https://github.com/xyd-js/examples
cd examples/icons
``` 

3. Run dev server:
```bash [descHead="Info" desc="Above command runs a dev server"]
xyd
```

3. Enjoy!
::::
