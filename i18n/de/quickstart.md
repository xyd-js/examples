# Schnellstart

Starten Sie den Entwicklungsserver:

```bash
xyd
```

Sie finden:

- `/quickstart` — Englisch (Standardsprache)
- `/pl/quickstart` — Polnisch
- `/de/quickstart` — Deutsch

Die Seitenleiste jeder Sprache stammt aus dem entsprechenden `navigation.languages[]`-Eintrag in `docs.json`.
