# Quickstart

Run the dev server:

```bash
xyd
```

You'll find:

- `/quickstart` — English (default locale)
- `/pl/quickstart` — Polish
- `/de/quickstart` — German

Each locale's sidebar comes from its own `navigation.languages[]` entry in `docs.json`.
